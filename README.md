# csvlens-pro

CSV explorer and lightweight visualizer.

## ✨ Feature
Preview CSV rows and column stats.

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run

```bash
python src/main.py
```

## 🧪 Test

```bash
python -m unittest discover
```

## 🚀 GitHub Upload

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/csvlens-pro.git
git push -u origin main
```
